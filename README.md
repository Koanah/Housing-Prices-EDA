### Housing Prices EDA

This project performs exploratory data analysis on the Ames Housing dataset.
It includes data cleaning, visualization, and a baseline regression model. This project was done and run fully on google collab


## Requirements
- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn

## How to Run
1. Download the dataset from (https://www.kaggle.com/c/house-prices-advanced-regression-techniques/data )
2. Upload `train.csv` and `test.csv` to Google Drive
3. Open the notebook in Google Colab
4. Run all cells from top to bottom
